# RetirementIQ — macOS App

Native WKWebView wrapper for the Mac App Store.

## Build pipeline

1. `./build.sh` — builds web bundle to `retiq-deploy/app/`
2. `cp -r retiq-deploy/app/. macos/App/RetirementIQ/app/` — copies bundle into Xcode project
3. Open `macos/App/RetirementIQ.xcodeproj` in Xcode
4. Archive → Distribute App → App Store Connect

## Xcode project setup (one-time, manual)

The Xcode project file is not generated by script. To set it up:

1. Open Xcode → New Project → macOS → App
2. Set product name: RetirementIQ
3. Bundle identifier: app.retirementiq.mac
4. Language: Swift, no storyboards (delete Main.storyboard, set NSPrincipalClass)
5. Replace generated AppDelegate.swift and ViewController.swift with the ones in this repo
6. Add the `app/` folder as a **folder reference** (drag into Xcode, select "Create folder references")
7. Set deployment target: macOS 13.0
8. Set signing: Automatically manage signing, Team BJW727236S
9. Set entitlements file to RetirementIQ.entitlements
10. Enable Hardened Runtime (required for MAS)
